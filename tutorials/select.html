Understanding the SELECT Statement

The SELECT statement is the most fundamental SQL command, used to retrieve data from a database. It allows you to specify exactly which columns you want to see and from which tables.

Basic Syntax

SELECT column1, column2, ...
FROM table_name;

To select all columns from a table, you can use the asterisk (*) wildcard character:

SELECT * FROM table_name;

Key Concepts

  • Columns: Specify which fields you want to retrieve
  • Table: The source of your data
  • Semicolon: SQL statements end with a semicolon

Examples

Example 1: Selecting Specific Columns

Let's say we have a table called users with columns for id, name, email, and age. To retrieve just the name and email:

Query

SELECT name, email 
FROM users;

Result

name email
John Doe john@example.com
Jane Smith jane@example.com
Bob Johnson bob@example.com

Example 2: Selecting All Columns

To retrieve all columns from the users table:

Query

SELECT * 
FROM users;

Result

id name email age
1 John Doe john@example.com 28
2 Jane Smith jane@example.com 34
3 Bob Johnson bob@example.com 42
